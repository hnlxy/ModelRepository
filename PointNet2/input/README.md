# 输入数据说明

此文件夹用于存放推理预测时需要的输入数据。

## 数据格式要求

对于点云分类任务，输入数据应为：
- 文件格式：.txt 或 .npy
- 数据格式：每行包含 x, y, z 坐标，可选包含法向量 nx, ny, nz
- 点云数量：建议 1024 个点

## 示例文件结构
```
input/
├── README.md (此文件)
├── sample_point_cloud.txt
└── batch_inference/
    ├── cloud1.txt
    ├── cloud2.txt
    └── ...
```

## 使用方法

1. 将要预测的点云文件放入此文件夹
2. 运行推理脚本进行预测
3. 结果将保存到 output 文件夹中
